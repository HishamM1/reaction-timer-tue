# reaction-timer
Reaction Timer Mini Game with Vue.js using Vue hooks, components, props, and custom events to emit it in the child components.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
